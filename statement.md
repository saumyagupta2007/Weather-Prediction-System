Project Statement – Weather Prediction System

This project is a Weather Prediction System developed using Python without using any machine learning libraries such as sklearn, tensorflow, or pytorch. The main purpose of this project is to predict basic weather conditions like Sunny, Rainy, and Cloudy based on input parameters such as temperature, humidity, and wind speed.

Instead of applying complex machine learning algorithms, this system uses a distance-based mathematical approach to find similarities between user input data and historical weather dataset values. The system calculates the distance between input values and predefined weather categories, and the category with the minimum distance is selected as the final prediction.

The project demonstrates how logical thinking and mathematical formulas can be used to simulate real-world prediction systems without actual machine learning models. It helps in understanding how data-driven decisions can be made using simple Python programming concepts.

The main objective of this project is to provide a strong foundation for beginners in the field of data science and artificial intelligence, by showing how datasets can be processed and analyzed without relying on external libraries.

This project is useful for learning:

- How to work with CSV datasets using Python
- How classification can be performed without ML libraries
- How mathematical distance formulas are applied in prediction systems
- How basic AI-like logic can be implemented using programming
- How real-world problems can be solved using simple algorithms


Overall, this project acts as a beginner-friendly simulation of a weather forecasting system, helping students understand the core concepts behind prediction systems and preparing them for advanced topics in Machine Learning and Artificial Intelligence.
